# Liberty (Свободен)

[![Android](https://img.shields.io/badge/Platform-Android-green?logo=android)](https://developer.android.com/)
[![Kotlin](https://img.shields.io/badge/Kotlin-2.0-blue?logo=kotlin)](https://kotlinlang.org/)
[![Compose](https://img.shields.io/badge/Jetpack-Compose-4285F4?logo=jetpackcompose)](https://developer.android.com/jetpack/compose)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**Liberty** — это приватный трекер привычек с открытым исходным кодом, созданный для тех, кто стремится к свободе от нежелательных зависимостей. Приложение ориентировано на максимальную конфиденциальность: ваши данные зашифрованы и никогда не покидают ваше устройство.

## ✨ Ключевые особенности

*   **📊 Прогресс в реальном времени**: Отслеживание серий (стреков) с точностью до часа.
*   **🔐 Безопасность военного уровня**: База данных зашифрована с помощью **AES-256 (SQLCipher)**.
*   **☝️ Биометрический замок**: Защита входа по отпечатку пальца или лицу.
*   **📓 Журнал срывов**: Анализ триггеров и ведение записей для работы над собой.
*   **🏆 Система достижений**: Игровые механики для мотивации на длинной дистанции.
*   **👥 Семейный режим**: Поддержка нескольких профилей с отдельными PIN-кодами.
*   **🌓 Адаптивный дизайн**: Поддержка светлой, тёмной и системной тем (Material 3).
*   **📥 Экспорт/Импорт**: Создание бэкапов в формате JSON.
*   **🏠 Виджеты**: Мониторинг главной привычки прямо с рабочего стола.

## 🛠 Технологический стек

*   **UI**: Jetpack Compose + Material 3
*   **DI**: Hilt (Dagger)
*   **Database**: Room + SQLCipher (шифрование)
*   **Async**: Kotlin Coroutines & Flow
*   **Navigation**: Compose Navigation + Type-safe arguments
*   **Charts**: Vico (современные графики)
*   **Worker**: WorkManager (ежедневные уведомления)
*   **Widgets**: Jetpack Glance
*   **Local Storage**: DataStore Preferences

## 🚀 Начало работы

### Системные требования
*   Android 8.0 (API 26) или выше.
*   Android Studio Ladybug (или новее).
*   JDK 17.

### Сборка
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/svoboden.git
   ```
2. Откройте проект в Android Studio.
3. Дождитесь завершения Gradle Sync.
4. Запустите приложение на эмуляторе или физическом устройстве.

## 🛡 Безопасность и Конфиденциальность

1.  **Локальное хранение**: Приложение не имеет доступа к интернету. Все данные хранятся в `/data/data/com.svoboden.app/databases/`.
2.  **Шифрование БД**: Ключ шифрования генерируется при первом запуске и безопасно хранится в Android Keystore.
3.  **Скрытие контента**: При переходе в фон содержимое приложения скрывается в диспетчере задач, если включена биометрическая защита.
