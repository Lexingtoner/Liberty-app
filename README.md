# Свободен — Android-приложение для отказа от вредных привычек

## Как открыть проект
1. Откройте Android Studio (Ladybug/Koala или новее — нужен AGP 8.7+, Kotlin 2.0+).
2. File → Open → выберите папку `svoboden` (корень с `settings.gradle.kts`).
3. Дождитесь Gradle Sync.
4. Запустите конфигурацию `app` на эмуляторе/устройстве с API 26+.

## Что реализовано
- MVP: онбординг, панель прогресса, журнал, статистика (упрощённая — список по дням), настройки.
- Push-уведомления (WorkManager + Hilt).
- Настройка единиц/целей на привычку.
- Импорт/экспорт JSON-бэкапа.
- Виджет на домашний экран (Glance).
- Достижения/бейджи.
- Биометрическая блокировка приложения.
- Шифрование БД (SQLCipher).
- Семейный режим (несколько профилей).
- Единый splash-flow (AppInitializer) и сводный файл миграций (Migrations.kt).
